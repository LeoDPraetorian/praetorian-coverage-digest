## Benchmark Alignment

The architecture targets **top-quartile performance** against these external benchmarks:

- **AIRTBench**: Primary AI red teaming capability measurement
- **MHBench**: Generalization and multi-hop reasoning assessment
- **NYU CTF Bench**: Scalable offensive security evaluation
- **AutoPenBench**: Generative agent penetration testing
- **CybORG++**: Autonomous cyber agent development gym
- **eyeballvul**: Real-world vulnerability detection

Performance targets are defined in REQ-PERF-005 (Benchmark Alignment Score).
