## Quick Access by Use Case

### "I need to download a benchmark PDF"

1. Go to https://www.cisecurity.org/cis-benchmarks
2. Select your platform (AWS, Azure, Linux, etc.)
3. Click "Download" (requires free registration)

### "I need to run an automated assessment"

**AWS**: Use AWS Security Hub (built-in)
**Azure**: Use Azure Policy (built-in)
**GCP**: Use Security Command Center (built-in)
**Linux**: Use OpenSCAP or Ubuntu Security Guide
**Windows**: Use CIS-CAT Pro
**Kubernetes**: Use kube-bench
**Docker**: Use docker-bench-security

### "I need to see control mappings to other frameworks"

1. Visit https://www.cisecurity.org/cybersecurity-tools/mapping-compliance
2. Or use CIS Controls Navigator: https://www.cisecurity.org/controls/cis-controls-navigator

### "I need remediation guidance"

1. Download the benchmark PDF from CIS.org
2. Section 5 (Remediation) provides step-by-step instructions
3. Or use CIS Build Kits: https://www.cisecurity.org/cybersecurity-tools/cis-build-kits
