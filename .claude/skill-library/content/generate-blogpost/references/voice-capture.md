# Voice Capture - Multi-Dimensional Style Analysis

**Goal**: Extract author's unique writing style from reference blog posts.

## Analysis Dimensions

### 1. Tone & Energy
- Word choice (formal vs conversational)
- Sentence structure
- Reader relationship (peer / teacher / expert)
- Energy level

### 2. Technical Depth
- Assumed knowledge level
- What gets explained vs assumed
- Use of analogies
- Theory vs practice balance

### 3. Structure & Flow
- Narrative patterns (problem-first / journey / deep-dive)
- Section organization
- Transition techniques
- Section length preferences

### 4. Personality & Humor
- Humor style and frequency
- Self-aware commentary
- Cultural references
- Meta-commentary patterns

### 5. Code Examples
- Inline vs block examples
- Annotation style
- Real vs simplified code
- Integration with prose

## Voice Profile Template

Document each dimension with verbatim examples from reference posts.

## Application Strategy

- Keep voice profile visible during drafting
- Check against profile after each section
- Watch for "generic blog voice" creep
- Compare marked sections to references
